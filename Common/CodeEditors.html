Плагины для вашего редактора

Рекомендуемые плагины для редакторов кода

Note: Дублирование описания плагинов сделано для того, чтобы человек сразу видел полную информацию о плагине для интересующего его редактора.

Atom

  • [Common] ESLint - линтинг js-кода;
  • [Common] EditorConfig - конфигурация настроек редактора через конфигурационный файл (полезно для синхронизации настроек в рамках проекта между членами команды);
  • [React] React JSX - поддержка JSX синтаксиса;
  • [TypeScript] Atom TypeScript - поддержка TypeScript в Atom;
  • [TypeScript] TSLint - линтинг ts-кода;

VS Code

  • [Common] ESLint - линтинг js-кода;
  • [Common] EditorConfig - конфигурация настроек редактора через конфигурационный файл (полезно для синхронизации настроек в рамках проекта между членами команды);
  • [TypeScript] TSLint - линтинг ts-кода;

Sublime Text 3

  • [Common] Package Control - плагин для управления плагинами;
  • [Common] ESLint - линтинг js-кода;
  • [Common] EditorConfig - конфигурация настроек редактора через конфигурационный файл (полезно для синхронизации настроек в рамках проекта между членами команды);
  • [React] ReactJS - поддержка JSX синтаксиса;
  • [TypeScript] TypeScript - поддержка TypeScript в Sublime Text 3;
  • [TypeScript] TSLint - линтинг ts-кода;

Vim (Neovim)

  • [Common] vim-plug - плагин для управления плагинами;
  • [Common] Neomake - линтер для всего;
  • [Common] Deoplete - автокомплит для всего;
  • [Common] NERDTree - удобная навигация по файловой системе;
  • [Common] EditorConfig - конфигурация настроек редактора через конфигурационный файл (полезно для синхронизации настроек в рамках проекта между членами команды);
  • [JavaScript] vim-javascript - поддержка js в vim;
  • [React] ReactJS - поддержка JSX синтаксиса;
  • [TypeScript] tsuquyomi - поддержка TypeScript;
